Localhost Automate just got updated!

Hello fellow Automates!

After reading a review on the Localhost Automate page over at the Chrome Web Store, I realized the extension should have the multiple localhost addresses feature implemented. Even though a developer using multiple localhost addresses is an edge case, it doesn't mean you guys should be ignored. ;)

This new version of Localhost Automate allows you to have multiple localhost addresses.
This means that in addition to the default 'localhost' entry you have from the previous version, you can now use different folders for different localhost locations (including ports) in your machine.

For example, when using multiple localhost entries, your Localhost Automate popup could look like this:


In a case like this, you will get redirected, depending on the 'Localhost directory' field, to the relevant location:

  • the 1st entry would take you to the default localhost address (e.g. http://localhost)
  • the 2nd entry will take you to the default localhost address as well, only with a different port (e.g. http://localhost:8080)
  • and the 3rd entry will take you to to http://localfolder2 .

You have complete freedom! :)
